A note on sizing

The primary means of scaling button size is by changing the font-size. Any size-modifying classes are set in percentages thereof.

Classes:

Base Class

.button sets the base styling for all buttons.

Using .button on its own will give you a button with all the default styling

Size Classes

standard

The default sizing is as follows:

Property Value Global Variable
height & line-height 2.5em $bh-button-height
padding (right & left) 2em HARD-CODED

Note the usage of ems... this allows resizing by changing the font-size.

extended

.extended increases the right and left padding to 3em.

x-extended

.x-extended increases the right and left padding to 4em.

fit-width

.fit-width makes the button display: block and stretches it to the width of its parent

small

.small decreases the height to 2ems and the horizontal paddings to 1em

jumbo

.jumbo increases the height to 3ems
