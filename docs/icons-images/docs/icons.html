Icons

Add solid vector icons without adding content to the page.

Classes

icon

.icon adds the base classes to show 'an' Icon in the :before pseudo-element.

icon-{{glyph-name}}

This is how a specific icon is selected. A listing of icons available can be found here.
(This will set the content of the :before pseudo-element to the char-code representing that glyph in the generated icon-font.)

Example

1
<span class="icon icon-airplane"></span>

{{platform}}-icon

.desktop-icon and .mobile-icon do the same as .icon, but only for the specified platform

Example

1
<span class="icon-mobile icon-airplane"></span>

icon--after

.icon--after will add the base classes to the :after pseudo-element.
Note: This will not work with .icon-{{glyph-name}} and you will need to use other means of inserting the glyph char-code (see the mixins below).

icon-accessible

Depracated... don't use it.

icon modifiers

  • .rotate-icon-left and .rotate-icon-right will turn an icon on its side.
  • .flip-icon will rotate an icon 180 degrees.
  • .stretch-icon will make an icon the size of its first non-static-positioned parent.

Mixins

bh-icon-rules

Add the base classes to show an Icon in an element.

1
2
3
.some-selector:before{
  @include bh-icon-rules;
}

bh-icon-after

Place an icon in a :after element.

1
2
3
.some-selector{
  @include bh-icon-after($glyph-name);
}
Parameter Type Default Value Description
$glyph-name string REQUIRED an icon name
$add-icon-rules boolean true set to false if you're using this together with the .icon-after class

icon-rotator

Rotate an icon
works for standard :before icons only

1
@include icon-rotator($rotation);
Parameter Type Default Value Description
$rotation CSS <angle> REQUIRED An angle representing the rotation delta

Example

Create a diagonal chevron.

1
<span class="icon icon-arrow-down-light diagonal-chevron"></span>


1
2
3
.diagonal-chevron{
  @include icon-rotator(-45deg);
}
