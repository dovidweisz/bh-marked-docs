Sprite

Streamlined image spriting

Sprite doesn't provide any global classes.

It imports a sass-map of sprite information from /client/sass/sprites, and exposes mixins (and a function) to implement sprite images.

For information on editing and adding sprite images see the documentation in /client/sass/sprites

Functions

With the exception of the sprite-layout mixin, all mixins require a sprite-data map that can be retrieved using a function

get-sprite-data

Get a sass-map inluding the data for a named sprite.

1
get-sprite-data($sprite-name);

Properties of the returned map:

Key Description
dimensions A map listing the individual images and their "dimentions"
img-url The path to the sprite image within the client directory.

Properties of a dimensions map:

Key Description
left Negative positioning in pixels for this image in the sprite
top Negative positioning in pixels for this image in the sprite
width Width for this image within the sprite
height Height for this image within the sprite

There currently is no get-dimensions-data function. To retrieve the dimentions you need to use the sass built-in map-get

Mixins

sprite-layout

Create all the styles needed for a complete sprite.
This mixin is meant to be an implementation example. Feel free to use it as a reference to create your own implementation

1
@include sprite-layout($sprite-name);
Parameter Type Default Description
$sprite-name string REQUIRED The name of a sprite defined in the $sprites hash

sprite

Everything an individual image needs besides for the url.
combines sprite-base, and sprite-dimensions (below)

1
@include sprite($sprite-dimensions, $display);
Parameter Type Default Description
$sprite-dimensions sass-map REQUIRED A sprite-dimentions map
$display Css <display> block element display property

sprite-base

Base Sprite styles (not related to a specific one)

1
@include sprite-base($display);
Parameter Type Default Description
$display Css <display> block element display property

sprite-dimensions

Set the dimensions for an image
Combines sprite-position, sprite-height, and sprite-width (below)

1
@include sprite-dimensions($sprite-dimensions);
Parameter Type Default Description
$sprite-dimensions sass-map REQUIRED A sprite-dimentions map

sprite-position

Set the background-position for an image 

1
@include sprite-position($sprite-dimensions);
Parameter Type Default Description
$sprite-dimensions sass-map REQUIRED A sprite-dimentions map

$sprite-dimensions sass-map REQUIRED A sprite-dimentions map

sprite-width

Set the width for an image 

1
@include sprite-width($sprite-dimensions);
Parameter Type Default Description
$sprite-dimensions sass-map REQUIRED A sprite-dimentions map
