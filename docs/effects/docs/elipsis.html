Elipsis

Add elipsis to trunkated multiline text.

There are no classes because each element needs it's own implementation

Mixins

bh-ellipsis

Set the height of a container and add elipses when the texts gets bigger than the container.

1
@include bh-ellipsis( $height, $child-selector, $background-color)
Parameter Type Default Value Description
$height number REQUIRED Height of the container in pixels
$child-selector string .bh-ellipsis-inner css selector for inner wrapping element
$background-color color $white background-color to be set on the elipses to mask text wrapping element underneath.

Example:

1
2
3
4
5
<p class="product-description">
    <div class="bh-ellipsis-inner">
        Product description text of variable length
    </div>
</p>


1
2
3
4
.product-description{
    background-color: $white;
    @include bh-ellipsis(32);
}

If you can't set the class for the inner element, and your background color is not white you need to specify more parameters

1
2
3
4
5
.product-description{
    $background-color: $bnh;
    background-color: $background-color;
    @include bh-ellipsis(32, '.product-description-inner', $background-color);
}

bh-ellipsis-hide

To remove the effect (on-hover etc.)

1
@include bh-ellipsis-hide;

Example:

1
2
3
4
5
6
7
.product-description{
    @include bh-ellipsis(32);
    &:hover{
        height: initial;
        @include bh-ellipsis-hide;
    }
}
