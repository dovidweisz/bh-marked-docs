Hover Transition

Easily define transitions for hover

Mixin

Add transitions for hover.

1
@mixin hover-transition($transition-to, $transition-property, $transition-time)
Parameter Type Default Description
$transition-to Number or Color REQUIRED The animatable property value you to transition to on hover
$transition-property property name color the property you want to transition
$transition-time duration 300ms the transition duration.
