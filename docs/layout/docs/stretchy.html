Stretchy

Use absolute positioning to fill containers and achieve pixel perfect layouts.

(Where the container has a fixed height and width.)

source

Classes

bh-stretchy

.bh-stretchy fill 100% of the positioned parent.

Parameter Type Default Description
$offset number or list 0 Offset (in pixels) from the positioned parent
$exclude-dirs... keyword(s) UNDEFINED Sides not to set an offset for

Mixin Examples

Simple

Margins

Add 25 pixel margins (from the positioned parent) to all sides

1
@include bh-stretchy(25);

Add 10 pixel margins to the right and left

1
@include bh-stretchy(0 10);

Use negative positioning to make this element bigger than the container.

1
@include bh-stretchy(-25);

.article{
    //settings
    $_left-nav-width: 250;
    $_right-add-width: 400;
    $_margins: 10;
 
    //.article styles
    width: 100%;
    height: rem-calc(2000);
 
    &-main{
        $_left: $_left-nav-width + $_margins * 2;
        $_right: $_right-add-width + $_margins * 2;
 
        @include bh-stretchy($_margins $_right $_margins $_left);
    }
    &-navigation{
        @include bh-stretchy($_margins, right);
        width: $_left-nav-width;
    }
    &-add-bar{
        @include bh-stretchy($_margins, left);
        width: $_right-add-width;
    }
}
