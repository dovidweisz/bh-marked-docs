Waterfall css and sass framework

Source for global css styles, sass mixins, and sass variables

About the framework

This framework is based on Zurbs Foundation (for sites) framework.

Besides for the global styles generated by foundation, their entire suite of mixins is available for use in our own scss code.

Index

Directories

  • component - this and utils should be refactored in to one directory
  • global - Contains the entry-points that generate the global csss
  • ng1 - Lagacy global styles. These should eventually be moved to their respective components.
  • platforms - platform build overrides (made possible by dynamically setting the sass-include-paths)
  • settings - sass variable setters for Foundation sass as well as our own.
  • sprites - generated file that is meant to be committed to the repo. todo: rename to make this more obvious
  • tasks - build tasks and plugin settings
  • temp - this file is generated code not in the repository
  • utils - Utility mixins and classes

Files

  • app.scss This is the entry point for sass (mixins and variables - importing this does not output any css)
  • foundation-app.scss this is the entry point for foundations framework
  • gulpfile.js Build tasks to generate icon-fonts and global css
