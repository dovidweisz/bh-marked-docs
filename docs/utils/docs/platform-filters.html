Platform Filters

Tools for differentiating between platforms in sass and css.

There are two variables that can tell you what platform is being rendered, $bh-is-desktop and $bh-is-mobile. But the better way to use them is by using these mixins and classes.

Classes

Hiding

These classes all apply display: none !important; when their conditions are met:

bh-show-for-mobile
bh-hide-for-desktop
bh-show-for-desktop
bh-hide-for-mobile

Floating

These classes will float elements for the named platform:

bh-desktop-float-left
bh-mobile-float-left
bh-desktop-float-right
bh-mobile-float-right

Mixins

bh-mobile-only and bh-desktop-only allow you to write a block of code that will only be rendered for that specific platform.
For example, if you want an element to have the outer-glow effect, but not on mobile.

.my-special-component{

    font-weight: $global-weight-semi-bold;
    @include bh-desktop-only{
        @include outer-glow;
    }
}
