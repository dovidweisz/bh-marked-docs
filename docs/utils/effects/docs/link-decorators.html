Link Decorators

Decorators for links, and link-like elements

Classes

bh-ui-link

.bh-ui-link Makes an element act like a button/link

  • Sets the color to $whale (for inactive, :active, and :hover states)
  • Removes the underline text-decoration
  • Sets the display to inline-block

Mixins

link-color

Set the color for inactive, :active, and :hover states. (overiding the reset)

1
@include link-color($color, $hover-active-color);
Parameter Type Default Value Description
$color color REQUIRED The color for the in-active state
$hover-active-color color same as $color The color for :hover and :active states

bh-ui-link

  • Calls link-color
  • Removes the underline text-decoration
  • Sets the display to inline-block
1
@include bh-ui-link($color, $hover-color)
